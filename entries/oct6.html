These allow for responsive webpages. The @media line (media query) can be called for several different widths to accommodate different screen sizes the webpage might be viewed on.

We also did a small workshop on creating a responsive grid based of a twelve-column layout. Each of the columns would increase by about 8%, where the twelfth column would be 100% of the screen width. Creating rows in between the different columns used allowed for multiple rows to be used. The variation of column widths did not matter except that they all had to add up to 100% for the row. This method of splitting up a webpage and making it responsive could prove very useful in to future once our projects become more and more advanced.
